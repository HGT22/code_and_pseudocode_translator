# Release v0.1.2

## ✨ Highlights
- First public release of the multilingual code translator.
- Universal AST + lexer + parser + transpiler architecture.
- CLI workflow for selecting source/target languages.
- Extended language coverage including pseudocode variants.
- Initial multiplatform build support.

## 🌍 Supported Platforms
- Windows
- macOS
- Linux
- Android
- iOS
- ChromeOS (via Linux/Web flow)

## 🧠 Supported Languages
- C, C++, C#, Objective-C, Objective-C++
- Python, Java, JavaScript, TypeScript, Rust, Swift, Ruby
- Go/Golang, Kotlin, PHP, R, Scala, Dart, Haskell, Elixir, F#
- SQL, MATLAB, D, Assembly (x86/x64/ARM), WebAssembly (WAT)
- Pseudocódigo orientado a C / Java / Python
- Pseudocodi orientat a C (URV style in output and normalization)

## ✅ Included in this release
- Language normalization improvements for real-world input styles.
- Catalan pseudocode support (URV-oriented C pseudocode flow).
- Battery tests for translation and language aliases.
- README direct download links for latest release assets.

## ⚠️ Notes
- If an asset is missing, keep the same filename strategy in the next release.
- If you rename assets, update download links in README.
